# ModelSaber Browser
An electron app that shows modelsaber so you can install models from an app PogU.

# Developing

Dependancies:
Electron,
Electron Packager

First,
`git clone https://github.com/parapass/modelsaber-browser/`
then,
`npm install`
finally,
`npm start`
